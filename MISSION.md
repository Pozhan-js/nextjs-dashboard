# Mission: PostgreSQL for this Next.js dashboard

## Why
Understand and safely operate the PostgreSQL database behind this Next.js dashboard on Vercel and Neon, so database setup, querying, and deployment are no longer black boxes.

## Success looks like
- Explain the roles of PostgreSQL, Neon, Vercel, and the `postgres` npm package
- Read the project's basic tables and SQL queries
- Connect the application through `POSTGRES_URL` without exposing credentials
- Recognize when constraints, transactions, and indexes matter

## Constraints
- Learn through the existing dashboard code rather than an unrelated demo
- Keep each lesson short and practical

## Out of scope
- Database administration at scale
- Advanced query planning and replication
