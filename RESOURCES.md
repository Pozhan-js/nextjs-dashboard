# PostgreSQL Resources

## Knowledge

- [PostgreSQL tutorial](https://www.postgresql.org/docs/current/tutorial.html)
  Official introduction to relational concepts and SQL. Use for foundational study.
- [PostgreSQL constraints](https://www.postgresql.org/docs/current/ddl-constraints.html)
  Official reference for `NOT NULL`, `UNIQUE`, primary keys, and foreign keys. Use when designing tables.
- [PostgreSQL transactions](https://www.postgresql.org/docs/current/tutorial-transactions.html)
  Official explanation of all-or-nothing database operations. Use when several writes must succeed together.
- [PostgreSQL indexes](https://www.postgresql.org/docs/current/indexes-intro.html)
  Official introduction to query acceleration and index costs. Use when a real query becomes slow.
- [Connect an application to Neon](https://neon.com/docs/connect/connect-from-any-app)
  Official guide to Neon connection strings, pooled connections, SSL, and `DATABASE_URL`-style environment variables.
